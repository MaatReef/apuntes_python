5. Crear la Tabla de Tareas

        Los modelos son como construimos tablas de bases de datos.. En Django.
        Para cada tabla debemos crear una clase.. Que represente a una tabla.
        Mientras que sus atributos vendrías a ser como sus columnas o campos.        

        
            from django.db import models
            from django.contrib.auth.models import User

            class Tarea(models.Model):
                usuario = models.ForeignKey(User, 
                                on_delete=models.CASCADE,
                                null=True,
                                blank=True)
                titulo = models.CharField(max_length=200)
                descripcion = models.TextField(null=True,
                                                blank=True)
                completo = models.BooleanField(default=False)
                creado = models.DateTimeField(auto_now_add=True)

                def __str__(self):
                    return self.titulo

                class Meta:
                    ordering = ['completo']
        

        Ahora debemos enviar está informacion a la base de Datos, HACER EL MIGRATE.
            python manage.py makemigrations

        De esta manera ya migramos y existe dentro de nuestro archivo 0001_initial.py
            python manage.py migrate

        También debemos registrar desde admin.py.. Que se encuentra dentro del proyecto base.
        
            from django.contrib import admin
            from .models import Tarea

            # Register your models here.
            admin.site.register(Tarea)
        

        Ahora migramos al sitio.. Lo podemos ver al correr el servidor y observarlo desde el admin.
            python manage.py runserver

15. Restringir Acceso

        Ahora usamos el LoginRequiredMixin que provee Django..
        
            from django.shortcuts import render
            from django.views.generic.list import ListView
            from django.views.generic.detail import DetailView
            from django.views.generic.edit import CreateView, UpdateView, DeleteView    
            from django.contrib.auth.views import LoginView
            from django.contrib.auth.mixins import LoginRequiredMixin       #Importamos el mixin que provee Django para autenticar.
            
            
            from django.urls import reverse_lazy
            from .models import Tarea
            
            class Logueo(LoginView):
                template_name = "base/login.html"
                field = '__all__'
                redirect_authenticated_users = True
            
                def get_success_url(self):
                    return reverse_lazy('tareas')
            
            
            class ListaPendientes(LoginRequiredMixin, ListView):        #Agregamos la restrincción donde deseemos.
                model = Tarea  
                context_object_name = 'tareas'
            
            class DetalleTarea(LoginRequiredMixin, DetailView):         #Agregamos la restrincción donde deseemos.
