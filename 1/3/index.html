6. Diccionarios

        Colecciòn de pares de claves, cada clave tiene relacionada un determinado valor.
        Dichas claves son ùnicas..

        mi_dic = { 'c1':'valor1', 'c2':'valor2', 'c3':'valor3' }

        Podemos buscar en los diccionarios en base a su clave.. 
        No tiene un indice, un orden especifico.
        El orden es irrelevante, buscamos valores en base a su clave.

        Los diccionarios, util para guardar data que se relacione con otro concepto.
        Caracteristicas de una persona por ejemplo.

        --------------------------------------------------------

        diccionario = {'c1':'valor1','c2':'valor2'}

        print(type(diccionario))	-> 

        --------------------------------------------------------

        diccionario = {'c1':'valor1','c2':'valor2'}

        diccionario = {'c1':'valor1','c2':'valor2'}

        print(diccionario)

        resultado = diccionario['c1']
        print (resultado)		-> valor1

        --------------------------------------------------------

        cliente = {'nombre':'Juan', 'apellido': 'Fuentes', 'peso': 88, 'talla': 1.76}
        consulta = (cliente['apellido'])

        print(consulta)		-> Fuentes

8. Sets

        Colecciòn de elementos, como las listas, con alguna que otra diferencia.

        Se pueden declarar como
            set([1,2,3,4,5,6])	
        o bien
            {1,2,3,4,5,6}		->Python diferencia con los diccionarios.
        
        mi_set = (['a','b','c'])

        Python descarta los valores duplicados, los descarta
        
        Ademàs los elementos dentro de un set, no estàn ordenados en indices..
        Y sus elementos son inmutables.
        
        No se pueden agregar listas y diccionarios.
        --------------------------------------------------------
                    -> El conjunto de llaves puede ser [] o {}, o bien ()
                    -> Ya que cada set espera al menos un argumento..

        print(mi_set)			-> {1,2,3,4,5}

9. Booleanos - Bools

        Dos valores, True o False

        Declararlo de manera directa..
        mi_bool_1 = True
        mi_bool_2 = False

        Declararlo de Manera Indirecta.. Python interpreta.
        mi_bool_1 = 5 > 4	
        mi_bool_1 = 5 < 4

        < 	menor que 	
        > 	mayor que	
        >= 	mayor o igual que 	
        <= 	menor o igual que 	
        == 	igual que 	
        != 	no igual que

        Al usar este tipo de operadores de manera indirecta usaremos booleanos

        Tambièn ademàs si consultamos al encontrar o no determinados valores.

        mi_bool = 5 in mi_lista
        mi_bool = 5 not in mi_lista
        --------------------------------------------------------
        *De manera directa
        var1 = True	-> Debe colocarse con mayùscula.
        var2 = False

        print(type(var1))	-> < class 'bool'>
        print(type(var1))	-> True
        print(type(var2))	-> < class 'bool'>
        print(type(var2))	-> False
        --------------------------------------------------------
        *De manera indirecta
        numero = 5 > 2+3

        print(type(numero))	-> False 

        --------------------------------------------------------

        numero = 5 == 2+3

        print(type(numero))	-> True


        numero = 5 >= 2+3

        print(type(numero))	-> True


        numero = 5 != 2+3

        print(type(numero))	-> False 

        --------------------------------------------------------
        *Se puede tambièn expresar asì

        numero = bool(5>6)

        print(numero)		->False

        *Tambièn
        numero = bool()

        print(numero)		->False
        --------------------------------------------------------

        lista = [1,2,3,4]
        control = 5 in lista

        print(type(control))		->< class 'bool'>
        print(control)			->False

        lista = [1,2,3,4]
        control = 5 not in lista

        print(type(control))		->< class 'bool'>
        print(control)			->True


        --------------------------------------------------------
