14. Proyecto del Dìa

        El programa primero debe preguntarle el nombre al usuario.
        El programa interactua con el usuario, invitandolo a elegir un número entre el 1 al 100, teniendo solo 8 intentos para adivinarlo..
        En cada intento el usuario dirá un número y el programa responderá, cuatro cosas distintas.
        A - Si el número que ingresa el usuario es menor a 1 o mayor a 100, el programa responde que no es un número posible.	
        B - Si el número elegido por el usuario es menor al que ha elegido el programa, el programa responde que su respuesta es incorrecta y que ha elegido un número 

        menor al numero secreto.
        C - Si el usuario eleige un número mayor, también el programa se lo hará saber.
        D - Si el usuario acierta el número secreto, se le avisa que ha ganado y cuantos intentos le ha tomado.

            -> Si pierde, le dará la opción de volver a participar en un nuevo juego con otro número
            -> Esto hasta que el usuario gane o pierda los 8 intentos.

        from random import randint

        intentos = 0
        estimado = 0
        numero_secreto = randint(1,100)
        nombre = input("Dime tu nombre: ")

        print(f"Bueno {nombre}, he pensado un número entre 1 y 100\nTienes 8 intentos para adivinar")

        while intentos < 8:
            estimado = int(input("Cuál es el número?: "))
            intentos += 1

            if estimado < numero_secreto:
                print("Mi numero es mas alto")
            elif estimado > numero_secreto:
                print("Mi numero es mas bajo")
            else:
                print(f"Felicitaciones {nombre}! Has adivinado en {intentos} intentos")
                break

        if estimado != numero_secreto:
            print(f"Lo siento, se han agotado los intentos. El numero secreto era {numero_secreto}")
