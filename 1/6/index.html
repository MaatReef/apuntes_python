3. Crear y Escribir Archivos

        Ahora tocarà como hacer cambios en el archivo original, en el anterior ejemplo utilizabamos una variable para ayudarnos.

        --------------------------------------------------------
        De acuerdo a los modos de apertura tendremos
        
        mi_archivo = open('archivo.txt', 'r')			-> Modo solo lecutra, sin posibilidad de realizar modificaciones en el archivo.
                                    -> Esta opción viene por defecto.
        
        mi_archivo = open('archivo.txt', 'w')			-> Modo solo escritura, si el archivo ya existe es vaciado, se resetea a cero
                                    -> Si no existe el archivo, lo crea.
        
        mi_archivo = open('archivo.txt', 'a')			-> Modo solo escritura posicionandose al final del archivo.
                                    -> Se crea en caso de que el archivo no exista
                                    -> Si existe, se posiciona al final del documento, para poder modificarlo a partir de allí.
        
        --------------------------------------------------------
        
        
        archivo = open('prueba.txt', 'r')		-> Error en la r, lectura, debemos abrirlo en escritura
        
        archivo.write('soy el nuevo texto')		
        
        archivo.close()
        
        --------------------------------------------------------
        
        archivo = open('prueba1.txt', 'w')			-> Escritura, si el archivo tiene texto, lo reemplazamos con el nuevo.	
        
        archivo.write('soy el nuevo texto')			-> El archivo tendrá este nuevo contenido a partir de ahora, sobreescribiendo lo anterior.
        
        archivo.close()
        
        --------------------------------------------------------
        
        *Para el salto de línea colocar el \n
        archivo.write('soy el nuevo texto\n')			
        O bien
        archivo.write('''Hola
        mundo
        aqui
        estoy''')			-> otra manera de incorporar el salto de línea.
        --------------------------------------------------------
        
        archivo = open('prueba.txt', 'w')
                                        -> Poco uso, si se tiene una lista de strings, para escribir en diferentes líneas
                                        -> Mejor usar un loop for con el método write, para escribir cada elemento separado con un 								salto de linea
        archivo.writelines(['hola', 'mundo', 'aqui', 'estoy'])		-> Algo confuso, porque en realidad escribe todos los strings uno atras de otro.
                                        -> holamundoaquiestoy		
        archivo.close()
        
        --------------------------------------------------------
        
        archivo = open('prueba.txt', 'w')
        
        lista =['hola', 'mundo', 'aqui', 'estoy']
        
        
        for p in lista:					-> Evitamos el uso anterior y lo mejoramos a través del ciclo con writelines, en fin, poco práctico.
            archivo.writelines(p + '\n')
                                                
        archivo.close()
        
        --------------------------------------------------------
        *Usado el 'a', para agregar al final, de esta manera incluimos un log, un registro de un programa que quedará registrado en un archivo. 
        
        archivo = open('prueba.txt', 'a')		-> Con w, borraría el contenido anterior y con r no podriamos, solo lectura.

5. Pathlib

        Es un componente importante de la biblioteca estandar de python que se utiliza para manipular rutas de sistemas de archivos en cualquier sistema operativo.  
        ->pathlib se incorpora en la versión 3.4 de python
        
        En el anterior punto lo hacíamos sin pathlib, ya que antes no se empleaba se hacía de manera más rústica.
        Entonces ahora al incorporarlo..
        --------------------------------------------------------
                        -> En este caso no usamos ni el open ni el close como anteriormente si.
                        -> Es decir, pathlib lo hace más "sencillo"
        from pathlib import Path
        
        carpeta = Path("C:/Users/Win10/Desktop/Python/Dia 6/prueba.txt")
        print(carpeta.read_text())	-> Vemos el contenido del archivo prueba.txt
        
        
        print(carpeta.name)		-> extrae el nombre del archivo	-> prueba.txt
        
        
        print(carpeta.suffix)		-> .txt
        
        
        print(carpeta.stem)		-> prueba		->Podemos entonces diseccionar el archivo.
        
        
        --------------------------------------------------------
        
        from pathlib import Path
        
        carpeta = Path("C:/Users/Win10/Desktop/Python/Dia 6/prueba.txt")
        
        if not carpeta.exists():			-> Verificamos con existes() si el archivo existe o no.
            print("Este archivo no existe")
        else:
            print("Genial, existe")			-> Genial, existe
        
        --------------------------------------------------------

6. Path

        Es una de las clases màs importante de Pathlib.
        Cuando creamos una instancia de la clase Path.. Lo hacemos para representar una ruta a un archivo o un directorio, en el sistema de archivos de nuestra pc.
        Por lo tanto es util para..
        -Crear o mover Archivos
        -Enumerar Archivos			-> Que coincidan con una extensión o un patrón determinado.
        -Crear rutas basadas en strings		
        
        Path puede pasar cualquier colección de strings a un formato de ruta de acceso. 
        Por ejemplo..
        
        mi_ruta = Path('Europa','España','Barcelona','Sagrada_Familia.txt')
        
        print(mi_ruta)		-> Europa/España/Barcelona/Sagrada_Familia.txt
                        -> Como jerarquía de carpetas.
        
        --------------------------------------------------------
        
        from pathlib import Path
        
        guia = Path("Barcelona", "Sagrada_Familia")
        print(guia)					-> Barceloan\Sagrada_Familia 	-> No es una ruta de acceso pero es lo que hace Path, construye objetos.
                                                -> Objetos que responden al formato de una ruta de Acceso. 
        
                -> El objeto Path asignado a la variable guia, contiene una ruta relativa.
                                    -> Si fuese una ruta absoluta, serìa una ruta especìfica.
        
        --------------------------------------------------------
        from pathlib import Path
        
        
        base = Path.home()
        guia = Path(base, "Barcelona", "Sagrada_Familia")
        
        print(base)				-> Generamos así el Home, C:\Users\Win10
                            -> Es una instancia absoluta la que devuelve home()
        
        print(guia)		-> C:\Users\Win10\Barceloan\Sagrada_Familia	-> si le pasamos la base primero con el home, imprimimos la ruta completa
        
        
        
        --------------------------------------------------------
        * El constructor Path, acepta tanto cadenas, como objetos preexistenes de Path
        
        from pathlib import Path
        
        base = Path.home()
        guia = Path(base, "Europa", "España", Path("Barcelona", "Sagrada_Familia.txt"))		-> Path interno
        
        print(guia)		-> C:\Users\Win10\Europa\España\Barcelona\Sagrada_Familia.txt
                            -> De esta manera el constructor Path gestiona de manera "inteligente" ambos tipos de objetos
                            -> Uniéndolos de manera correcta, agregándole la barra de acuerdo al sistema operativo.
        
        --------------------------------------------------------
        
        from pathlib import Path
        
        base = Path.home()
        guia = Path(base, "Europa", "España", Path("Barcelona", "Sagrada_Familia.txt"))		
        
        guia2 = guia.with_name("La_Pedrera.txt")
        
        print(guia)		-> C:\Users\Win10\Europa\España\Barcelona\Sagrada_Familia.txt
        
        print(guia2)		-> C:\Users\Win10\Europa\España\Barcelona\La_pedrera.txt		-> Apunta correctamente con with_name
                                                        -> Creamos una 2ª ruta cambiando el destino.
        
        --------------------------------------------------------
        
        from pathlib import Path
        
        base = Path.home()
        guia = Path(base, "Europa", "España", Path("Barcelona", "Sagrada_Familia.txt"))		
        
        print(guia.parent)	-> C:\Users\Win10\Europa\España\Barcelona		-> El parent trae al padre, el antecesor más inmediato de una ruta determinada.
        
        
        print(guia.parent.parent)		-> C:\Users\Win10\Europa\España
        
        print(guia.parent.parent.parent)		-> C:\Users\Win10\Europa
                                            -> Así seleccionamos una porción de la ruta.
        
        --------------------------------------------------------
        *También podremos usar la clase path, para enumerar archivos dentro de un árbol de carpetas.
    
        
        
        Europa		
        -> España	-> Barcelona		-> La pedrera.txt
                            -> Sagrada familia.txt
        
                   Idem		-> Madrid
                
        ->		-> Francia	-> Paris		-> Torre Eiffel.txt
        
        ->		-> consejos.txt
        
        ->		-> Normativas.txt
        
        
        Para enumerar todos los archivos del directorio Europa
        
        from pathlib import Path
        
        guia = Path(Path.home(),"Europa")
        
        for text in Path(guia).glob("*.txt"):
            print(text)				
                    -> C:\Users\win10\Europa\Consejos.txt
                    -> C:\Users\win10\Europa\Normativas.txt		-> Devuelve los dos archivos.txt, del directorio Europa.
                    
        --------------------------------------------------------
        
        from pathlib import Path
        
        guia = Path(Path.home(),"Europa")
        
        for text in Path(guia).glob("**/*.txt"):	->De manera recursiva usamos el método glob, para encontrar todos los archivos .txt del Directorio Europa
            print(text)				-> Incluyendo sus subs directorios.
        
                    ->C:\Users\win10\Europa\Consejos.txt		
                    ->C:\Users\win10\Europa\Normativas.txt		
                    ->C:\Users\win10\Europa\España\Barcelona\La_Pedrera.txt		
                    ->C:\Users\win10\Europa\España\Barcelona\Sagrada_Familia.txt
                    ->C:\Users\win10\Europa\España\Madrid\Museo_del_Prado.txt
                    ->C:\Users\win10\Europa\Francia\Paris\Torre_Eiffel.txt		
        
        --------------------------------------------------------
            También podemos calcular rutas relacionadas entre sí, con relative to.. Cuandos se quiere recuperar una porción de archivos largos.
        
        from pathlib import Path
        
        guia = Path("Europa", "españa", "Barcelona", "Sagrada_Familia.txt)
        
        en_europa = guia.relative_to(Path("Europa"))				-> Desde Europa hacia "abajo" armamos la ruta que se desee.
        en_espania = guia.relative_to(Path("Europa", "España"))			-> Desde España hacia "abajo"
        
        print(en_europa)		-> España\Barcelona\Sagrada_Familia.txt 
        print(en_espania)		->Barcelona\Sagrada_Familia.txt
        
        
            -> Por lo cual relative_to, devuelve un nuevo objeto Path, relacionado con el argumento determinado.
